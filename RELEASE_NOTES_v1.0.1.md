## Summary

Public release polish for the repository.

This release adds a repository homepage README, clarifies the public-facing structure, and keeps both the production Agnes bundle and the reusable bearer-token template in one publishable project.

- `skills/agnes-image-generator-hermes-refresh`
  - production-ready Agnes Image 2.1 Flash skill bundle

- `templates/bearer-token-image-skill-template`
  - reusable template for bearer-token image APIs
  - provider/model/endpoint/env-key placeholders for fast adaptation
